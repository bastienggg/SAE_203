/backoffice

C'est ici que vous placerez l'application permettant d'administrer vos données.

La base de code qui vous est donnée correspond à la correction de R2.13-TP7/backoffice.

La structure de cette partie est volontairement simple car a priori suffisante pour les besoins de la SAE2.03.
Vous disposez de : 
    - /asset pour mettre vos images par exemple
    - /index.html, /style.css, /script.js pour votre HTML/CSS/JS

Si toutefois vous voulez utilisez des composants comme pour le site /client, libre à vous d'adpater la structure de projet.
